# Seed Connect: PR/FAQ (Public Release & Internal Strategy)

## 1. プレスリリース（2027年1月1日）
**【報道関係者各位】**

### 作品を消費する時代を終わらせる。クリエイターの”人となり”にスポットを当てた新プラットフォーム『Seed Connect』始動。

Seed Connect 運営事務局は本日、クリエイティブの「結果」ではなく、クリエイターの**「人となり」を価値の源泉とする新プラットフォーム『Seed Connect』**を正式に公開しました。

私たちは、既存のSNSが加速させてきた、文脈を無視した「作品の孤立化」とバズ消費に終止符を打ちます。本作は、「Seed（過程）」から伸びる「Tree（成長の軌跡）」と紐付いていない「Fruit（作品）」の存在をシステムレベルで許さない、有機的なデータ構造を持つプラットフォームです。

---

### ■ 開発の背景：アルゴリズムの「点」から、人間性の「線」へ
現代のコンテンツプラットフォームにおいて、創作はアルゴリズムによる「バズ」や「効率的な消費」に支配されています。そこでは、作品そのものの価値以上に、「流行の波に乗れるか」「瞬間的に目を引くか」という市場適合性が生存の鍵となっています。

その結果、クリエイターは「自分が何を表現したいか」よりも「どうすればアルゴリズムに選ばれるか」を優先せざるを得ず、キャッチーな断片だけが市場を独占する構造が固定化されました。勇気を持って踏み出した「はじめの一歩」や、葛藤の末に産み落とした作品でさえも、視聴回数や「いいね」の数といった数字だけで冷徹に選別され、その背景にある試行錯誤は無視されてしまう。そんな数値至上主義の構造が、多くの貴い挑戦を、誰にも気づかれないまま静かに押し潰しています。

しかし、創作の本質的な価値は、決して「市場に消費されやすいこと」だけではないはずです。その作品の裏側に、どんな思いや葛藤があったのか、どんな壁にぶつかり、どう乗り越えようとしたのか。私たちは、市場が切り捨てるそのプロセスに滲み出る**「人となり」**こそが価値の根源であり、何者にも真似できない掛け替えのないものであると信じています。

Seed Connectは、この「人となり」を共有することで、技術の未熟さや流行との乖離を「弱み」ではなく、深い共感を生む「価値」へと反転させる場所を目指しています。単なる作品投稿サイトではなく、何者でもない人が、流行や数字の呪縛から解き放たれ、自らの歩みを肯定してくれる熱量あるファンと繋がるための、新しい表現の聖域です。

---

### ■ Seed Connectの独自性：人となりを可視化する「3つの設計」

#### ① 「人となり」が伝わるジャーニー型UI
作品を閲覧する際、ユーザーは同時に、その作品へと至る時間軸（Tree）を旅することができます。深夜のボイスメモや、葛藤を綴った短いテキスト。それらが連なり枝分かれしながら、一つの「人となり」を形作っていく様を、独自のインターフェースで表現しました。

#### ② 「孤立した作品」の投稿制限
プロセスを持たない作品の投稿を物理的に制限することで、ユーザーに「成果物の消費」ではなく「人間への共鳴」を促します。ここは「上手い作品」を探す場所ではなく、「応援したい人」と出会う場所です。数字だけで選別されないよう、データ構造そのものから「背景」を不可分なものとして設計しました。

#### ③ 「未完成」を肯定する文化の醸成
「完璧でなければ出してはいけない」という呪縛からクリエイターを解放します。完成した時の「上手さ」ではなく、試行錯誤のプロセスこそを、人間性が最も滲み出る尊い表現として肯定します。未完成のままの苦悩さえも、誰かの心に届く価値へと変えていく文化を、プラットフォーム全体で守り抜きます。

不思議なことに、一番反響があったのは完成作（Fruit）ではなく、その裏にあった『どうしてもこの表情が描けない』という葛藤のメモでした。『上手い絵』ではなく『私という人間』を見てくれているという安心感で、数年ぶりに筆が止まらなくなりました。」  

> **「視聴維持率」の数字に削り取られていた映像作家（30代）**

「YouTubeだと、少しでも退屈だと思われるとすぐにブラウザバックされてしまうので、本当は残したかった『静かな空気感』も泣く泣くカットして、テンポ重視の編集ばかりしていました。 
Seed Connectでは、本編に入れられなかったカットや、撮影中に考えていたことのメモを並行して共有しています。ファンの方から『カットされた部分を見て、このシーンの意図がわかった』とコメントをもらった時、自分のこだわりを捨てずに済んだ気がして、救われました。」

作品が完成するまでの数ヶ月間、彼女がどんな壁にぶつかり、どう乗り越えたかを見てきたから、完成した曲を聴いた時の感動が桁違いでした。ただの視聴者ではなく、彼女の歩みを隣で見守ってきた『目撃者』になれた。こんなに深い音楽体験は初めてです。」

## 2. 外部向けFAQ（想定される質問と回答）

**Q: 既存のnoteやYouTube、SNSと何が違うのですか？**

**A:** noteは「記事（知見）」、YouTubeは「動画（娯楽）」といったコンテンツ（何を作ったか）が主役です。一方、Seed Connectはクリエイター自身（どんな人か）が主役です。一つのジャンルに特化せず、音楽も、日々の悩みも、工作も、すべてを「その人の物語」として統合して見せる点に特化しています。また、当サービスは、「未完成」や「失敗」を認める文化をシステムレベルで推奨しています。クオリティで競うのではなく、等身大の試行錯誤を見せることで、少人数でも熱量の高いファンと繋がるためのプラットフォームです。  

  
**Q: なぜ「人となり」に特化する必要があるのですか？**

**A:** AIが瞬時にコンテンツを生成できる時代において、最後に残る価値は「誰が、どんな想いでそれを作ったのか」という人間的な文脈だと信じているからです。スキルの優劣で競うのではなく、唯一無二の「人となり」で繋がることで、クリエイターが数字に魂を削られることなく、健やかに活動を続けられる場所を目指しています。  

  
**Q: 作品のクオリティが低くても投稿して良いのですか？**

**A:** はい。Seed Connectにおいて、クオリティの定義は「上手さ」ではなく「誠実さ」です。**むしろ「完成していないもの」や「試行錯誤」こそが、その人の人間味を伝える最も重要なコンテンツだと考えています。** 完璧な作品を見せる場所は他に譲り、ここは「あなた自身」をさらけ出す場所にしてください。今日始めたこと、今日悩んだこと、その一歩一歩のログ自体が、あなただけの「人となり」を形成する価値ある資産になります。  

  
**Q: 「未熟さをさらけ出す」ことは、プラットフォーム全体のコンテンツの質を低下させませんか？**

**A:** 私たちは**「質の低さ」を推奨しているのではなく、「質の追求プロセス」を肯定しています**。既存プラットフォームが「完成品の陳列棚」だとしたら、**Seed Connect は「アトリエ（工房）」**です。アトリエで泥臭く試行錯誤する姿を見せることは、完成品の価値を貶めるものではなく、むしろ作品に「かけがえのない物語」という付加価値を上乗せするものだと考えています。  

  
**Q: アルゴリズムに頼らず、どうやって新規参入者と視聴者をマッチングさせるのですか？**

**A:** 私たちは「再生数」や「流行」といった一律のスコアで評価するアルゴリズムを使いません。代わりに、ユーザーの「価値観」や「応援したい属性（例：〇〇に初挑戦中、〇〇の壁に苦戦中）」に基づいた、共鳴軸のレコメンドを実装します。大勢に消費されることではなく、あなたの歩みを心から愛してくれる「一人の熱狂的なサポーター」と出会うための仕組みを構築します。  

  
**Q: プロセスを公開すると、アイデアを盗まれたり、模倣されたりしませんか？**

**A:** むしろ逆です。Seed Connectに制作過程を記録することは、**そのアイデアが「いつ、誰の手によって、どのように生まれたか」という動かぬ証拠（タイムスタンプ）になります。** 完成品だけを出す世界では結果だけが盗用されやすいですが、試行錯誤のプロセスという「替えのきかない文脈」を公開しているあなたこそが、真のオリジナルであると証明されます。  

  
**Q: 創作の裏側を見せることは、作品の「神秘性」や「ブランド」を損なわないでしょうか？**

**A:** 私たちは、すべてのプロセスをさらけ出すことを強制はしません。どの「Seed」を公開し、どの「Tree」を育てるかはクリエイターの自由です。ただし、現代においてファンが求めているのは「遠くの神様」ではなく「共に歩む人間」としての誠実さです。ブランドを壊すのではなく、ブランドの「根」を見せることで、より強固な信頼関係を築くことができると考えています。  

  
**Q: 制作過程を投稿する作業自体が、創作の負担になりませんか？**

**A:** 私たちは投稿するための「作業」を増やすつもりはありません。スマホのボイスメモ、スケッチの断片、思考のなぐり書きなど、創作の過程で自然と生まれる「愛おしい思考の断片（かけら）」をそのまま置けるような、軽量な投稿体験を設計しています。投稿するための立派な記事は必要はありません。あなたの呼吸の一部を、そのままSeedとして共有してください。  

  
**Q: 未完成のものを出すと、心ない批判や「アドバイス罪」に晒されるのが怖いです。**

**A:** Seed Connectは、数字で評価し合う「戦場」ではなく、制作過程を慈しむ「聖域」です。そのため、一般的なSNSのような「拡散」や「無記名の誹謗中傷」が起きにくい設計（例：プロセスへのリスペクトを前提とした共鳴ボタンや、クローズドなフィードバック機能など）を徹底します。あなたの試行錯誤は、批判の対象ではなく、共感の対象として守られます。

## 3. 内部/アンチFAQ（批判的な問いへの論理的な回答）

**Q: マネタイズはどう考えているのか？**

**A:** 広告モデルは一切採用しません。広告モデルは「PV数（バズ）」を至上命令にするため、私たちのコンセプトである「人となり」や「心理的安全」を破壊するからです。収益は、プラットフォーム手数料（15%前後を想定）と以下の3軸で構築します。
* **ジャーニー・サポーター（継続課金）：** クリエイターの特定の「目標」への挑戦を並走して見守る「権利」への支援。
* **プロセス・チア（都度課金/投げ銭）：** 「Seed（断片）」に対する、試行錯誤への敬意としてのチップ。
* **マイルストーン・ギフト（目標達成祝い）：** 作品完成や継続の節目に対するお祝い。


**Q: noteやYoutubeと比較して「機能が圧倒的に少ない」状態で、どうやって最初の10人を使い続けさせるのか？**

**A:** 機能の豊富さではなく、**「投稿に対する心理的安全性の高さ」**で勝ちます。他プラットフォームが「評価される場所」であるのに対し、Seed Connectは「さらけ出してもいい場所」です。最初の10人には、2人の開発者が自ら最初のサポーターとして徹底的に伴走し、「見守られている」という実感を機能以上に提供します。


**Q: ニワトリとタマゴの問題（クリエイターとファンの不足）に対してどう立ち向かうのか？**

**A:** 巨大なプラットフォームをいきなり作ろうとはしません。まずは「顔の見える最小単位のコミュニティ」を1つ定義し、そこを完全にハックします（特定のワークショップ受講生など）。オープンな集客は後回しにし、クローズドな環境で「人となりが可視化される喜び」を証明します。


**Q: インフラコストを2人の資金力でどう維持するのか？**

**A:** 最初からYouTubeと同じ土俵（高画質・長時間動画）には立ちません。MVPでは外部APIを使い、低解像度に絞ることでコストを制御します。価値の源泉は「画質」ではなく「文脈」にあるため、リッチな体験は収益化の目処が立ってから段階的に実装します。


**Q: アルゴリズムを使わずにマッチングできるのか？**

**A:** 過去の行動データではなく、**「今のモード（感情）」から「関わるべき人」を引き合わせる**マッチングを実装します。ユーザーの意志を直接トリガーにすることで、アルゴリズムのブラックボックス化を防ぎます。

**Q: クリエイターが「プロセスを投稿し続けること」に挫折しないか？**

**A:** クリエイターに「頑張らせない」UIを徹底します。10秒で送れるボイスメモや、ボツにした写真の「供養（リリース）」など、創作のついでに生まれる「断片」をそのまま置ける体験を構築します。


**Q: 半年後に100人しか集まらなかった場合、続ける意義はあるのか？**

**A:** あります。100万人の浅い消費ではなく、100人の「人生を変える共鳴」を優先します。**その100人が、Seed Connectのおかげで筆を折らずに済んだなら、このプラットフォームは「成功」です。** 2人の身軽さを武器に、その100人と向き合い続け、種火（Seed）を育てます。


**Q: 2名体制で心理的安全をどう守るのか？ 荒らし対策コストで潰れないか？**

**A:** 「監視」ではなく「構造」で防ぎます。「拡散（リポスト）」機能を排除し、コメント権限を「文脈を理解した層」に限定するなど、**「攻撃するコストが、応援するコストより高い」設計**を徹底します。2名でも回せる、自律的な浄化作用を持つ空間を目指します。
